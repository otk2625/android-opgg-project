# JAVAGG

### 개요
-  라이엇 게임즈에서 리그 오브 레전드의 API를 기반으로 하여 전적을 보여주는 전적 검색 사이트 op.gg를 모티브로 해서 따라 만들기 위한 프로젝트

### 기능
- 소환사 이름으로 검색시 전적검색
- 각 게임의 세부 게임 정보 
- 현재 챌린저 랭킹 구현
- 로그인, 회원가입 (페이스북 API 로그인 추가)
- 커뮤니티 기능 구현(게시물 CRUD)
- 댓글기능, 좋아요 기능
